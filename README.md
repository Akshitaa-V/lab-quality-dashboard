# Interactive Dashboard for Laboratory Quality Control Monitoring

## 📌 Overview
This project focuses on building an interactive dashboard to monitor quality control (QC) in clinical laboratory testing.

The dashboard helps identify:
- Out-of-range test values
- Device inconsistencies
- Trends over time
- Quality violations

## 🎯 Features
- Time-series analysis of lab results
- KPI metrics for test accuracy
- Heatmap for device violations
- Control funnel plots
- Rolling average comparison

## 🛠️ Technologies Used
- D3.js
- Data Visualization Techniques
- Data Cleaning & Processing

## 📊 Key Visualizations
- Heatmap (Device vs Month violations)
- KPI comparison charts
- Time-series plots
- Control limit analysis
- Bar charts for violations

## 🚀 Future Improvements
- Real-time data updates
- Interactive filters
- Drill-down analysis
